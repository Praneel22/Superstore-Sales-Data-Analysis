 # Superstore Sales Data Analysis

Business Problem:
Analyze retail sales performance to identify revenue trends,
customer behavior patterns, and product profitability insights.

Tools Used:
Python | SQL | Tableau | Pandas | Data Visualization

Key Outcomes:
• Identified top-performing regions and categories
• Improved understanding of profit vs sales trends
• Built executive-level interactive dashboard

## Project Overview
This project analyzes retail sales performance using Python, SQL, and Power BI,Tableau

## Tools Used
- Python (Pandas, Matplotlib)
- SQL
- Power BI
- Excel

## Key Insights
- Regional sales performance analysis
- Profit & loss trends
- Category-wise revenue insights

## Project Architecture
Raw Data → Data Cleaning (Python) → SQL Analysis → Tableau Dashboard → Business Insights
